也就是说，只要我们爬到了网页的源代码，那么这个网页展示出来的（甚至隐藏掉的）所有信息，我们只要从源码中筛选就能够获取。

所以一般爬虫的原理其实非常简单：

  1. 爬取源码
  2. 筛选信息

如何获取源代码

我们访问网页一般都是通过浏览器，而浏览器其实也就是一个程序而已。相应的，爬虫其实也是一个程序。

因此爬虫程序（脚本）其实就是把自己伪装成一个浏览器，然后跟浏览器一样向目标网页的服务器发送访问请求，只要骗过了服务器，那么服务器就会把源码返回给你，这样你就获得该网页的源代码了。

上面指的网页都是静态网页，不包括动态内容；源代码也只是HTML代码，不包括JS代码。爬取动态内容要更为复杂一些

API

如果你是一个互联网从业者，那么应该听过API 这个词。

API（Application Programming Interface,应用程序编程接口）是一些预先定义的函数，目的是提供应用程序与开发人员基于某软件或硬件得以访问一组例程的能力，而又无需访问源码，或理解内部工作机制的细节。——百度百科

简单来说，你可以把API 当做一个管道接口，当你连接上这个管道接口后，你就能从中获取一些数据信息（当然，你能获取哪些信息是它说了算）。

很多企业都会对外开放一些API 接口，通过这些接口，我们能更为快速、高效、便捷地获取天气、航班、股市等信息。（不只是这些公共信息，其实像QQ、微博、facebook 等都会开放一些API 接口，这样能让公司的业务更好地渗透到市场中去，也利于业务之间的合作）

显然，我们也可以通过爬虫程序来连接这些API 接口，以此来获取数据信息。

产品经理学Python&&爬虫（一）：为什么我要学Python

By Tracy Lee

发表于 2016-09-05

什么是Python？

Python 是一门优雅、明确、简单的编程语言，拥有丰富和强大的库，适合编写脚本与进行数据分析

为什么我要学Python？

我现在以及未来要做的都是产品，不是程序员，那么，为什么我要学Python 呢？

一、提升工作效率

我的编程启蒙老师——浙江大学的翁恺曾经说过：“重复7次以上的动作就应该写代码（脚本）来做”

在这个时代，作为一个互联网人，几乎时刻都离不开计算机（包括移动设备）。我们会通过计算机处理很多数据和文件，当数据和文件量大到一定程度，或者重复次数很多的时候，通过代码（脚本）来实现我们的需求，在一定程度上能够解放我们的工作量，提升我们的工作效率。

记录我的大二

By Tracy Lee

发表于 2016-08-30

大学又过去了一年，从大一到大二，这一年里自己想要记录的东西太多太多了，无论什么时候回忆起来，都觉得是那么珍贵

一、学生工作

过去一年里，我继续担任 学生党员“长波班”班长、院学生会办公室主任、党支部副书记。

这一年里工作责任变得更重了，但是收获与付出是成正比的。只是自己在这一年里更加清楚自己未来想要走的方向和道路了，所以决定大三的时候不再从事不必要的学生工作，把所有精力投入我爱的互联网里去。

二、勤奋蜂

我从大一开始决心要成为一名产品（PM）。我们学校虽然互联网相关的实验室、工作室很多，但是设立产品组的却很少，因此我很长一段时间都只能自己看书、上网、画原型···来学习产品的知识。显然那段时间的成长是非常缓慢的。

后来在一个机缘下，我接触到了“重庆你好勤奋蜂科技”的创始人、CEO——刘智鹏。他正在跟一群在职的、在校的 志同道合的人，致力于一个“校园知识共享”以及“校园人才挖掘”的项目 CampusSay校园说 。你可以把它理解成“校园版 在行+知乎”，但它还不只如此。

刘智鹏了解到我的情况后，把我引荐给了他的合伙人、勤奋蜂的CPO。我跟CPO聊了一个多小时后，双方都觉得挺不错的，于是，我加入了勤奋蜂，正式开始了我的产品之路。

By Tracy Lee

发表于 2015-08-02

大学就过去一年了，时间依旧过得那么快，还记得上大学之前就一直想象并期待着美好的大学生活，现在一年过去了，当初的未来变成了过去，还是应该回顾并记录下来的。好好总结一下那些好的与不好的，然后才能更好地规划接未来。
