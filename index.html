大三上学期创业比赛全记录

发表于 2017-01-21 | 分类于 我的大学 |

一转眼，大三上学期已经结束。这个学期做的最多的事情就是【打比赛】。

我所在的创业团队因为各种需要，会去全国各地打很多创业比赛。之前一直都是我们的大BOSS亲自上阵，然而这学期他去台湾交换了，所以打比赛路演答辩的任务就交给了我。

从16年9月到12月，我参加了5次大型比赛，除了第一次的互联网+重庆市赛我只负责答辩之外，其他4次都由我独立负责路演和答辩的全部工作。

互联网+ 重庆市赛 —— 银奖

这是我第一次参加创新创业比赛，在重庆理工大学举行。

互联网+ 算是大学生里的最高赛事之一了，虽然我不是路演者，但能参加答辩也挺开心的。知道了怎么修改PPT，知道了路演答辩是什么样子的，知道了该怎么回答评委的问题。

这次的路演时间是5分钟，然而我们还有一分半的视频，导致讲的时间很少，没有很好的把我们的项目阐述清楚，在这次比赛后我们重新调整了路演策略以及 PPT 结构，算是一个教训吧。

那么用 Python 呢？只需要一行

Python 代码简短明确，相对来说比较好入门，适合在课余和工作之余 的时间去学习，所以大家不用担心自己没有时间。

PS. 我无意比较各语言的好坏，我知道 C、C++ 的运行效率比 Python 高很多，我也相信有大牛能够通过其他语言只用一行代码来实现。在此只是简述 Python 简短易入门。

产品经理学Python&爬虫（一）：为什么我要学Python

发表于 2016-09-05 | 分类于 产品 |

什么是 Python？

Python 是一门优雅、明确、简单的编程语言，拥有丰富和强大的库，适合编写脚本与进行数据分析

为什么我要学 Python？

我现在以及未来要做的都是产品，不是程序员，那么，为什么我要学 Python 呢？

一、提升工作效率

我的编程启蒙老师——浙江大学的翁恺曾经说过：“重复7次以上的动作就应该写代码（脚本）来做”

在这个时代，作为一个互联网人，几乎时刻都离不开计算机（包括移动设备）。我们会通过计算机处理很多数据和文件，当数据和文件量大到一定程度，或者重复次数很多的时候，通过代码（脚本）来实现我们的需求，在一定程度上能够解放我们的工作量，提升我们的工作效率。

记录我的大二

发表于 2016-08-30 | 分类于 我的大学 |

大学又过去了一年，从大一到大二，这一年里自己想要记录的东西太多太多了，无论什么时候回忆起来，都觉得是那么珍贵

一、学生工作

过去一年里，我继续担任 学生党员“长波班”班长、院学生会办公室主任、党支部副书记。

这一年里工作责任变得更重了，但是收获与付出是成正比的。只是自己在这一年里更加清楚自己未来想要走的方向和道路了，所以决定大三的时候不再从事不必要的学生工作，把所有精力投入我爱的互联网里去。

二、勤奋蜂

我从大一开始决心要成为一名产品（PM）。我们学校虽然互联网相关的实验室、工作室很多，但是设立产品组的却很少，因此我很长一段时间都只能自己看书、上网、画原型···来学习产品的知识。显然那段时间的成长是非常缓慢的。

后来在一个机缘下，我接触到了“重庆你好勤奋蜂科技”的创始人、CEO——刘智鹏。他正在跟一群在职的、在校的 志同道合的人，致力于一个“校园知识共享”以及“校园人才挖掘”的项目 CampusSay校园说 。你可以把它理解成“校园版 在行+知乎”，但它还不只如此。

刘智鹏了解到我的情况后，把我引荐给了他的合伙人、勤奋蜂的CPO。我跟CPO聊了一个多小时后，双方都觉得挺不错的，于是，我加入了勤奋蜂，正式开始了我的产品之路。

记录我的大一

发表于 2015-08-02 | 分类于 我的大学 |

大学就过去一年了，时间依旧过得那么快，还记得上大学之前就一直想象并期待着美好的大学生活，现在一年过去了，当初的未来变成了过去，还是应该回顾并记录下来的。好好总结一下那些好的与不好的，然后才能更好地规划接未来。

一、学生工作

过去一年里，我担任了 学生党员“长波班”班长、院学生会办公室的干事，和党支部副书记。

我们举办过一些学生活动。其中比较成功的，是我们联合“重邮微校”、重庆市血液中心以及重庆市红十字会一起举办的“无偿献血、加入中华骨髓库”活动。活动三天，我们号召了828名师生共同参与无偿献血，共采集血液246980ml，创造了学校的记录。

学生工作让我变得更加细心、守规矩和有责任心了，同时也认识了很多朋友。

我觉得这些经历、成长和认识的朋友，才是人生中最重要的。
